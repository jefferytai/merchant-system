🚀 如何使用新界面

1. 搜索商户
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • 输入城市、商户类型、关键词
   • 点击"搜索"或"再搜索"
   • 卡片会以动画形式淡入显示

2. 查看商户详情
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • 每个卡片显示关键信息
   • 点击"展开详情"查看 LinkedIn、官网
   • 再次点击"收起详情"

3. 管理联系状态
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • 点击状态标记（○ / ✓）切换
   • 绿色 ✓ = 已联系
   • 灰色 ○ = 未联系
   • 状态自动保存

4. 操作商户
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • ✉️ 邮件图标 → 生成商务邮件
   • 🗑️ 删除图标 → 删除商户

5. 导出数据
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   • 点击"保存数据"按钮
   • 下载 JSON 文件（带时间戳）
